🎯 O que é Duke Revisão?

Duke Revisão é um projeto educacional que demonstra conceitos fundamentais de Programação Orientada a Objetos (POO) através de um exemplo prático: um sistema de loja virtual.

💡 Conceitos Demonstrados:
  • ✅ Encapsulamento (private/public)
  • ✅ Herança (extends)
  • ✅ Polimorfismo (interfaces e classes abstratas)
  • ✅ Abstração (classe abstrata Produto)
  • ✅ Tratamento de Exceções
  • ✅ Coleções e Streams
  • ✅ Design Patterns (Strategy, Template Method)
  • ✅ Enums (Type-safe constants)
