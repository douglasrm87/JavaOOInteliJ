🚀 COMO EXECUTAR:

  1. Clone o repositório:
     $ git clone https://github.com/douglasrm87/JavaOOInteliJ.git
     $ cd JavaOOInteliJ

  2. Compile o projeto:
     $ mvn clean compile

  3. Execute a aplicação:
     $ mvn spring-boot:run

  4. Acesse no navegador:
     http://localhost:8081

  5. Faça login:
     Usuário: drm
     Senha: 12345
